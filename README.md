A command-line based library management system using Python OOP and JSON persistence.

 Features
- Add, issue, return books
- Search by title or ISBN
- JSON-based file storage
- Logging using `logging` module
- Menu-based CLI

 Run the App

IT helps teachers or students to easily find out the book of the particular author they want in seconds.
